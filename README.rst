Install

Github
------------

  library(devtools)
  library(testthat)
  install_github('computational-metabolomics/structToolbox')

------------

   conda create -n structtoolbox bioconductor-structtoolbox -c conda-forge -c bioconda -c computational-metabolomics
   source activate structtoolbox
